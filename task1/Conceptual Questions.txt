Q1. What is the difference between a List<int> and a List<dynamic> in Dart? Why is it usually better to use a typed list like List<int>?

A List<int> hold integers only while List<dynamic> can hold values of any type (like strings, double or objects) which can cause error in runtime if we are not careful for example if we accidentally put in the wrong type it can cause problems so using List<int> is generally safer because it uses type safety. and it is more readable and maintain.

Q4. Describe in plain English what the for-in loop syntax does in Dart. How is it different from a traditional for loop with an index? When would you prefer one over the other?

for-in loop goes through each element in a list directly with out index and this is useful simple and more easier to read when we just want to process every value. traditional for loop gives us more control using index we can skip or access specific positions using index. we prefer for‑in when we only care about the values, and a traditional for loop when we need the index as well.
